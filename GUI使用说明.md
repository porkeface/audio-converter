# 音频转换工具 - GUI 使用说明

## 🚀 快速启动

### 方法一：双击启动（推荐）
直接双击项目根目录下的 `run_gui.bat` 文件即可启动图形界面。

### 方法二：命令行启动
```bash
cd D:\codeProject\audio-converter-demo
uv run python gui.py
```

## 📝 使用步骤

### 1. 选择输入文件
- 点击 **"选择文件"** 按钮
- 选择你的 `.ncm` 文件
- 程序会自动检测格式并显示在日志中

### 2. 选择输出格式
- 选择你想要的格式：**FLAC**、**MP3** 或 **WAV**
- 默认是 FLAC（无损音质）

### 3. 指定输出路径（可选）
- 程序会自动生成输出文件名（与输入文件同名，扩展名不同）
- 你可以点击 **"浏览"** 按钮自定义保存位置和文件名

### 4. 开始转换
- 点击 **"🚀 开始转换"** 按钮
- 转换过程会显示在日志区域
- 转换完成后会弹出提示框

## ⚠️ 注意事项

1. **FFmpeg 依赖**：如果要转换为 MP3 格式，需要安装 FFmpeg
   - 下载：https://ffmpeg.org/download.html
   - 将 ffmpeg.exe 放到系统 PATH 或项目根目录

2. **文件大小**：NCM 文件通常较大（几十MB到上百MB），转换需要一些时间

3. **输出格式**：
   - **FLAC**：无损音质，文件较大
   - **MP3**：有损压缩，文件较小（需要 FFmpeg）
   - **WAV**：未压缩，文件最大

## 🐛 常见问题

### Q: 启动时提示 "tkinter 不可用"？
A: tkinter 是 Python 标准库，通常自带。如果没有，请重新安装 Python 并勾选 "tcl/tk and IDLE"。

### Q: 转换 MP3 时失败？
A: 请确保已安装 FFmpeg，并且可以在命令行中运行 `ffmpeg` 命令。

### Q: 输出文件在哪里？
A: 默认与输入文件在同一目录，可以在 "输出路径" 中查看或修改。

### Q: 如何批量转换？
A: 当前版本不支持批量转换，可以多次选择文件转换。未来版本可能会增加此功能。
