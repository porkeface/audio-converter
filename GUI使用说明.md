# 音频转换工具 - GUI 使用说明

## 🚀 快速启动

### 方法一：双击启动（推荐）
直接双击项目根目录下的 `run_gui.bat` 文件即可启动图形界面。

### 方法二：命令行启动
```bash
cd D:\codeProject\audio-converter-demo
uv run python gui.py
```

## 📝 使用步骤

### 1. 添加输入文件（三种方式）

#### 方式一：点击按钮选择（支持多选）
- 点击 **"📁 选择文件"** 按钮
- 在文件选择对话框中，按住 `Ctrl` 或 `Shift` 键可以多选文件
- 点击"打开"确认选择

#### 方式二：直接拖拽文件到列表框（推荐！）
- 选中一个或多个 `.ncm` 文件
- 直接拖拽到程序的**文件列表框**中
- 松开鼠标，文件会自动添加

#### 方式三：拖拽到窗口
- 也可以直接拖拽文件到程序窗口任意位置

### 2. 管理文件列表
- **移除选中**：选中文件后点击"❌ 移除选中"或双击文件
- **清空列表**：点击"🗑️ 清空列表"移除所有文件

### 3. 选择输出格式
- 选择你想要的格式：**FLAC**、**MP3** 或 **WAV**
- 默认是 FLAC（无损音质）

### 4. 指定输出目录（可选）
- 程序会自动使用第一个文件的目录作为输出目录
- 你可以点击 **"浏览"** 按钮自定义输出目录

### 5. 开始批量转换
- 点击 **"🚀 开始批量转换"** 按钮
- 转换过程会实时显示在日志区域
- 转换完成后会弹出统计信息

## 🎯 功能特点

✅ **文件拖拽**：直接拖拽文件到列表框，超级方便  
✅ **批量转换**：一次选择多个文件，一次转换完成  
✅ **多格式输出**：支持输出为 FLAC、MP3、WAV  
✅ **实时日志**：转换过程实时显示，随时查看进度  
✅ **自动检测**：自动识别 NCM 格式  
✅ **元数据保留**：自动提取歌曲信息（歌名、艺术家、专辑）  
✅ **专辑封面**：自动提取专辑封面（如果可用）

## ⚠️ 注意事项

### 1. 拖拽功能依赖
- 需要 `tkinterdnd2` 库支持（已自动安装）
- 如果拖拽不可用，请运行：`uv add tkinterdnd2`

### 2. FFmpeg 依赖
- 如果要转换为 **MP3** 格式，需要安装 FFmpeg
- 下载：https://ffmpeg.org/download.html
- 将 `ffmpeg.exe` 放到系统 PATH 或项目根目录

### 3. 文件大小
- NCM 文件通常较大（几十MB到上百MB），批量转换需要一些时间
- 转换在后台线程进行，界面不会卡死

### 4. 输出格式说明
| 格式 | 特点 | 需要 FFmpeg |
|------|------|-------------|
| **FLAC** | 无损音质，文件较大 | ❌ 不需要 |
| **MP3** | 有损压缩，文件较小 | ✅ 需要 |
| **WAV** | 未压缩，文件最大 | ❌ 不需要 |

## 🐛 常见问题

### Q: 拖拽文件时显示红色禁止符号？
A: 请确保：
1. 已安装 `tkinterdnd2`：`uv add tkinterdnd2`
2. 拖拽到**文件列表框**内，而不是其他区域
3. 如果仍不行，使用"选择文件"按钮

### Q: 启动时提示 "tkinter 不可用"？
A: tkinter 是 Python 标准库，通常自带。如果没有，请重新安装 Python 并勾选 "tcl/tk and IDLE"。

### Q: 转换 MP3 时失败？
A: 请确保已安装 FFmpeg，并且可以在命令行中运行 `ffmpeg` 命令。

### Q: 输出文件在哪里？
A: 默认与输入文件在同一目录，可以在 "输出目录" 中查看或修改。

### Q: 如何批量转换？
A: 
1. 点击"选择文件"按钮，按住 Ctrl/Shift 多选
2. 或者直接拖拽多个文件到列表框
3. 点击"开始批量转换"

### Q: 转换速度慢？
A: 
- NCM 解密本身很快，但如果转码为 MP3 会比较慢
- 建议使用 FLAC 输出（不需要转码，速度最快）

如果遇到问题，可以：
1. 查看日志区域的错误信息
2. 检查输入文件是否为有效的 NCM 文件
3. 确保输出路径有写入权限
